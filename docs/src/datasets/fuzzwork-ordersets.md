# Fuzzwork Ordersets

[data.everef.net/fuzzwork/ordersets/](https://data.everef.net/fuzzwork/ordersets/) contains a mirror of the [market orders provided by Fuzzwork](https://market.fuzzwork.co.uk/api/).
This is similar to the [Market Order Snapshots](market-orders.md).

Produced by the [Sync Fuzzwork Ordersets](../commands/sync-fuzzwork-ordersets.md) command.

## Backfills

Historical ordersets provided by [Caden Hunter](https://evewho.com/character/97131472) are available in [data.everef.net/fuzzwork/ordersets/backfills/caden-hunter-fuzzwork-ordersets/](https://data.everef.net/fuzzwork/ordersets/backfills/caden-hunter-fuzzwork-ordersets/).
These backfills provide ordersets from dates prior to the automated sync.
